## Usage

1. Install the **uv** Python package manager (recommended), and install dependencies:

   ```sh
   brew install uv
   uv pip install -r requirements.txt
   ```

2. Run the backend:

   ```sh
   uvicorn app.main:app --reload --port 8000
   ```

3. Run the frontend:

   ```sh
   npm install
   npm run dev
   ```
